Shazam2Spotify is now Shazify

Shazam2Spotify was taken down by a DMCA notice from Shazam.
They want to avoid unauthorized applications that use their name and logo.

The extension is now up again with a new name and a new temporary logo.

Please feel free to contact me or react on Twitter if you have any suggestions regarding this.
